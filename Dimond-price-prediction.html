Predecting Dimonds Price using machine learning¶

This nootebook looks into using various Python-based machine learning and data science labraries in an attempt to build a machine learning model capable for predicting the price of dimonds according to their attributes.

we are going to take the follwing approch:

  1. Problem Defination
  2. Data
  3. Evaluation
  4. Features
  5. Modeling
  6. Exparimentation ## 1. Problem dafination in a statement,

    Analyze diamonds by their cut, color, clarity, price, and other attributes,and predicting the price if dimonds

    2. Data¶

    The original data came from the Kaggle repository.https://www.kaggle.com/shivam2503/diamonds

    3. Evaluation¶

    If we can reach 95% accuracy at predicting the price of dimonds during the proof of concept, we 'll persue the project.

    4. Features¶

    Creating data dictionary Index counter

carat Carat weight of the diamond

cut Describe cut quality of the diamond. Quality in increasing order Fair, Good, Very Good, Premium, Ideal

color Color of the diamond, with D being the best and J the worst

clarity How obvious inclusions are within the diamond:(in order from best to worst, FL = flawless, I3= level 3 inclusions) FL,IF, VVS1, VVS2, VS1, VS2, SI1, SI2, I1, I2, I3

depth depth % :The height of a diamond, measured from the culet to the table, divided by its average girdle diameter

table table%: The width of the diamond's table expressed as a percentage of its average diameter

price the price of the diamond

x length mm

y width mm

# make the funcation of scores
def fit_and_score(models,x_train,x_test,y_train,y_test):
    # set random seed
    np.random.seed(42)
     # make the dictonary for the model scores
    model_scores = {}
    # make the loop for model rotation
    for name,model in models.items():
        # fit the model
        model.fit(x_train, y_train)
        # fit score
        model_scores[name] = model.score(x_test, y_test)
    return model_scores
